# Shiftaker — Symmetric File Encryption Tool

![Banner](https://github.com/A01L/Shiftaker/blob/main/info.png)

**Shiftaker** — это графическое приложение (GUI) на Python для надежного шифрования и дешифрования файлов с использованием алгоритмов симметричного шифрования. Программа позволяет защитить любой файл паролем, превращая его в защищенный формат `.shtr`.

---

## 🛠 Как это работает

1.  **Шифрование:** При выборе файла и вводе ключа (пароля) программа шифрует содержимое файла. В метаданные создаваемого файла `.shtr` вшивается хеш пароля для последующей проверки.
2.  **Дешифрование:** Для восстановления данных необходимо выбрать зашифрованный файл и ввести тот же ключ. Программа сравнивает хеш введенного пароля с тем, что хранится в метаданных, и, если они совпадают, возвращает файлу исходный формат и содержимое.

---

## ✨ Основные возможности
* **Симметричное шифрование:** Использует библиотеку `cryptography` (Fernet) для обеспечения безопасности данных.
* **Контроль целостности:** Использование хеширования MD5 для проверки ключа доступа.
* **Удобный GUI:** Интерфейс построен на базе `Tkinter`, включает окно предпросмотра и текстовое поле для отображения содержимого файлов.
* **Безопасный формат:** Все зашифрованные файлы получают расширение `.shtr`.
